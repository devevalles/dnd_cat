Mira quins fitxers han canviat amb `git diff` i `git status`, genera tu mateix un missatge de commit curt i descriptiu en català que resumeixi els canvis, i executa:
1. `git add -A`
2. `git commit -m "<missatge que has generat>"`
3. `git push`

Després confirma-li a l'usuari que els canvis ja estan guardats i pujats, i digues-li el missatge de commit que has fet servir.
